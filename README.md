# Blinkit Landing Page Clone

A responsive Blinkit Landing Page clone built using **HTML, CSS, and Bootstrap 5**. This project replicates the modern user interface of Blinkit, focusing on responsive design, clean layout, and attractive user experience.

## 🚀 Features

* Responsive design for mobile, tablet, and desktop devices
* Modern and clean UI inspired by Blinkit
* Bootstrap 5 components and grid system
* Navigation bar with search functionality
* Product categories section
* Promotional banners and offers section
* Footer with useful links
* Fast loading and optimized layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
